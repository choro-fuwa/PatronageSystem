# 策略管理系统

基于Spring Boot + Vue 3的量化投资策略管理系统，提供策略创建、回测、监控、交易等全流程管理功能。

## 🌟 项目特色

- **完整的策略管理流程**：从策略创建到实盘交易的完整闭环
- **强大的回测引擎**：基于Python的量化回测，支持多种策略类型
- **实时监控预警**：多维度风险监控和智能预警系统
- **美观的现代化界面**：基于Element Plus的美观用户界面
- **模块化架构设计**：前后端分离，易于扩展和维护

## 🏗️ 系统架构

### 后端技术栈
- **Spring Boot 2.7.x**：主框架，提供RESTful API
- **MyBatis-Plus**：数据库ORM层
- **JPype 1.4.x**：Python-Java桥接，用于量化计算
- **MySQL/PostgreSQL**：数据存储

### 前端技术栈
- **Vue 3.x**：前端主框架
- **Element Plus**：UI组件库
- **ECharts 5.x**：数据可视化
- **Axios**：HTTP请求库
- **Vue Router**：路由管理

### 量化引擎
- **Python 3.8**：回测核心计算
- **pandas/numpy**：数据处理
- **TA-Lib**：技术指标计算
- **backtrader**：回测框架

## 🚀 快速开始

### 环境要求
- JDK 11+
- Node.js 16+
- Python 3.8+
- MySQL 8.0+

### 后端启动
```bash
cd patronage-strategy
mvn spring-boot:run
```

### 前端启动
```bash
cd patronage-ui
npm install
npm run dev
```

### Python依赖安装
```bash
pip install jpype1 numpy pandas backtrader ta-lib
```

## 📋 功能模块

### 1. 策略管理
- **策略创建**：支持4种策略类型（大类资产配置、FOF组合、基金指数、择时组合）
- **策略编辑**：参数配置、权重设置、再平衡规则
- **策略审核**：风控审核流程
- **策略复制**：快速复制现有策略

### 2. 回测系统
- **历史回测**：基于历史数据的策略验证
- **模拟运行**：基于实时数据的虚拟交易
- **压力测试**：极端市场情景测试
- **绩效分析**：多维度绩效指标计算

### 3. 监控预警
- **实时监控**：策略运行状态实时跟踪
- **风险指标**：波动率、VaR、最大回撤等
- **预警规则**：自定义预警条件和阈值
- **通知系统**：邮件、短信、系统内通知

### 4. 交易管理
- **订单管理**：交易指令生成和执行
- **持仓管理**：当前持仓查看和调整
- **再平衡**：自动和手动再平衡功能
- **交易记录**：完整的交易历史记录

### 5. 风险管理
- **风险概览**：整体风险状况展示
- **限额管理**：风险限额设置和监控
- **压力测试**：多情景风险测试
- **合规检查**：监管合规性检查

### 6. 报告中心
- **绩效报告**：策略绩效详细分析
- **风险报告**：风险指标和归因分析
- **归因分析**：收益和风险归因分解
- **对比分析**：与基准指数的对比

## 🔧 核心算法

### 策略计算算法
- **大类资产配置**：均值方差优化、风险平价模型
- **FOF组合管理**：基金筛选、权重优化、风险控制
- **基金指数组合**：指数化配置、跟踪误差最小化
- **择时策略**：技术指标信号、宏观经济指标

### 风险计算算法
- **波动率计算**：历史波动率、隐含波动率
- **VaR计算**：历史模拟法、参数法、蒙特卡洛法
- **最大回撤**：滚动窗口最大回撤计算
- **夏普比率**：风险调整后收益指标

### 绩效归因算法
- **收益归因**：资产配置贡献、标的选择贡献、择时贡献
- **风险归因**：边际风险贡献、风险集中度分析
- **风格分析**：规模因子、价值因子、动量因子
- **行业分析**：行业配置和集中度分析
